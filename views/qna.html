QnA

Here's some frequently asked question about HYOP:

Q: Is HYOP free?

A: Yes. HYOP is licensed under MIT License that makes it free and open-source.

Q: How I login? What's the password?

A: Default login username and password is admin and pass.

Q: Why I can't register? How I register a new user?

A: We don't allow users do self-registering for security reason. To registering a new user, go to the console and type register [username] [password] and press enter.

Q: How I install the app?

A: Go to the console and type install [app name] and press enter.
